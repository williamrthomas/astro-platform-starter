About Arcade Hub

Arcade Hub is a lightweight, framework-free platform for hosting HTML5 games and web applications. It's built with speed, simplicity, and beautiful UX in mind, using pure HTML, CSS, and JavaScript without any complex frameworks or build tools.

Our Mission

We believe that web games should be accessible, fast-loading, and enjoyable on any device. Our platform is designed to showcase the power of modern web technologies without the bloat of heavy frameworks or complex build processes.

Key Components

Main Platform

  • Landing page with game listings, filtering, and search functionality
  • Uses Tailwind CSS via CDN and Alpine.js for interactivity
  • Responsive design that works on mobile, tablet, and desktop

Game System

  • Currently includes games like Snake, Calculator, and Memory Match
  • Each game has its own directory with standalone implementation
  • Shared high score system and social sharing functionality

Development Tools

  • Python-based tools for game creation and management
  • game_helper.py for creating game scaffolds and registering games
  • game_idea_generator.py for generating game concepts

Documentation

  • Comprehensive design principles and standards
  • Detailed game development process
  • Game categorization system with metadata structure

Project Structure

The project follows a clean, organized structure with games, shared resources, documentation, and tools clearly separated.

Development Workflow

The platform has a well-defined workflow for adding new games:

  1. Generate a game idea (optional)
  2. Create a game scaffold using the helper tool
  3. Implement the game
  4. Register the game in the main registry
  5. Deploy to Netlify

Want to Contribute?

We welcome contributions from developers of all skill levels. Whether you want to create a new game, improve our platform, or fix bugs, check out our GitHub repository to get started.
